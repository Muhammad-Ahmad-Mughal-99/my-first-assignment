# GitHub Basics Assignment

## About This Project
This project was created to learn the basics of Git and GitHub.  
It includes understanding repositories, commits, branches, and pull requests.

## What I Learned
- What GitHub is and how it is used
- What a repository is (a project folder)
- How to make changes and save them (commit)
- How to use branches to work on separate versions
- How to create and merge pull requests

## Steps I Performed
- Created a repository on GitHub
- Added and edited a README file
- Created a new branch
- Made changes in the branch
- Created a pull request
- Merged the changes into the main branch

## Challenges I Faced
- Understanding how branches work
- Learning how pull requests and merging works

## Conclusion
This assignment helped me understand how developers manage code and collaborate using GitHub.
